AI Avatar UI

Drop-in web components for an interactive 3D avatar chat experience.

Usage

Load the bundle, then place a .avatar-shell container anywhere on your page:

<script type="module" src="https://YOUR-DEPLOYMENT-URL/ai-avatar-ui.js"></script>

<div class="avatar-shell" style="width: 480px; height: 640px;">
  <avatar-model backend="https://your-backend.example.com"></avatar-model>
  <avatar-status></avatar-status>
  <avatar-captions></avatar-captions>
  <avatar-inputs></avatar-inputs>
  <avatar-settings></avatar-settings>
</div>

Backend URL: pass your own backend's URL via the backend attribute on <avatar-model>. If you omit <avatar-model> entirely, this bundle auto-creates one for you inside .avatar-shell — but it'll default to calling same-origin (/ask, /settings, etc. on whatever domain the embedding page is served from), which only works if your backend shares that origin. Most integrators will want to include <avatar-model backend="..."> explicitly, as shown above.
