AI Avatar UI

Drop-in web components for an interactive 3D avatar chat experience.

Usage

Load the bundle, then include the components inside an .app-shell container — layout and theming are applied automatically once the script runs: 

<script type="module" src="https://ai-avatar-ui-ghost.vercel.app/ai-avatar-ui.js"></script>

<div class="app-shell">
  <div class="avatar-stage">
    <avatar-model backend="https://your-backend.example.com" avatar-scale="1" avatar-vertical-offset="-1.25"></avatar-model>
    <avatar-captions></avatar-captions>
  </div>
  <avatar-status></avatar-status>
  <avatar-settings></avatar-settings>
  <avatar-inputs></avatar-inputs>
</div>

Backend URL: set the backend attribute on <avatar-model> to your own backend's URL. See the project README's API Contract section for the routes it needs to implement.
